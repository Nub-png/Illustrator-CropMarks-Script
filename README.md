1. ОПИСАНИЕ
--------------------------------------------------------------------------------
Скрипт для автоматической расстановки типографских меток реза (Crop Marks) 
и меток совмещения (Registration Marks) в Adobe Illustrator.

Создан для подготовки макетов к печати. Позволяет быстро и точно расставить 
метки на выделенных объектах или артбордах без необходимости делать это вручную.


2. ВОЗМОЖНОСТИ
--------------------------------------------------------------------------------
• Crop Marks (метки реза) — короткие линии по углам объекта/артборда
• Registration Marks (метки совмещения) — круги для совмещения цветов в печати
• Работа с выделенными объектами
• Работа с активным артбордом
• Работа со всеми артбордами документа
• Отрицательное смещение — метки внутрь объекта
• Нулевое смещение — метки по краю объекта
• Выбор цвета меток из палитры (Registration, Black, C=100... и другие)
• Отдельный слой "myCropMarks" — метки не смешиваются с макетом
• Сохранение настроек между запусками
• Настройка длины, смещения, толщины линий и радиуса


3. УСТАНОВКА
--------------------------------------------------------------------------------
Способ 1 (рекомендуемый):
  1. Скопируйте файл CropMarks_AI.jsx в папку сценариев Illustrator:
     
     Windows:  C:\Program Files\Adobe\Adobe Illustrator [версия]\Presets\ru_RU\Сценарии\
     macOS:    /Applications/Adobe Illustrator [версия]/Presets.localized/ru_RU/Scripts/
  
  2. Перезапустите Illustrator.
  3. Скрипт появится в меню: Файл → Сценарии → CropMarks_AI

Способ 2:
  1. Откройте Illustrator.
  2. Выберите: Файл → Сценарии → Другой сценарий...
  3. Найдите и выберите файл CropMarks_AI.jsx.
  4. Скрипт запустится.


4. ИСПОЛЬЗОВАНИЕ
--------------------------------------------------------------------------------
4.1. Запуск
  1. Выберите объект(ы) или просто откройте документ.
  2. Запустите скрипт через меню: Файл → Сценарии → CropMarks_AI.
  3. В появившемся окне настройте параметры.
  4. Нажмите "OK".

4.2. Настройки диалогового окна
  • Crop Marks (метки реза) — включить/выключить
  • Registration Marks (метки совмещения) — включить/выключить
  • Длина — длина линии метки (мм)
  • Смещение — расстояние от края объекта/артборда до начала метки (мм)
    Положительное → метки снаружи
    Отрицательное → метки внутрь
  • Толщина — толщина линий меток (мм)
  • Радиус — радиус круга для Registration Marks (мм)
  • Цвет меток — выбор цвета из палитры документа
  • Область для меток:
    - Выделенные объекты — метки вокруг каждого выделенного объекта
    - Активный артборд — метки по краям активного артборда
    - Все артборды — метки по краям всех артбордов документа


5. НАСТРОЙКИ
--------------------------------------------------------------------------------
Скрипт запоминает последние использованные настройки в текстовом файле.
Путь к файлу:
  Windows:  C:\Users\[Имя пользователя]\Documents\Illustrator\Scripts\cropmarks_settings.txt
  macOS:    /Users/[Имя пользователя]/Documents/Illustrator/Scripts/cropmarks_settings.txt

Файл имеет простой текстовый формат:
  doCrop=true
  doReg=false
  length=3
  offset=3
  width=0.25
  radius=3
  color=Black
  area=0

Значения можно отредактировать вручную в любом текстовом редакторе.


6. ПРИМЕРЫ
--------------------------------------------------------------------------------
6.1. Стандартные метки для печати (наружу)
  Длина: 3 мм, Смещение: 3 мм, Толщина: 0.25 мм
  → Метки располагаются снаружи объекта/артборда

6.2. Метки внутри объекта (для вырубки)
  Длина: 3 мм, Смещение: -3 мм, Толщина: 0.25 мм
  → Метки располагаются внутрь объекта/артборда

6.3. Метки по краю объекта (без отступа)
  Длина: 3 мм, Смещение: 0 мм, Толщина: 0.25 мм
  → Метки начинаются прямо от края

8. УСТРАНЕНИЕ НЕИСПРАВНОСТЕЙ
--------------------------------------------------------------------------------
8.1. "Нет открытого документа"
  → Откройте документ в Illustrator перед запуском скрипта.

8.2. "Ничего не выделено"
  → В режиме "Выделенные объекты" необходимо выделить хотя бы один объект.
    Или переключитесь на "Активный артборд" или "Все артборды".

8.3. Метки не отображаются
  → Проверьте, включён ли слой "myCropMarks" и не находится ли он под другими слоями.
  → Проверьте цвет меток — если выбран Registration, убедитесь, что он есть в палитре.

8.4. Настройки не сохраняются
  → Проверьте наличие папки: 
    Windows: C:\Users\[Имя]\Documents\Illustrator\Scripts\
    macOS:   /Users/[Имя]/Documents/Illustrator/Scripts/
  → Если папки нет — создайте её вручную.

8.5. Скрипт не запускается
  → Убедитесь, что файл имеет расширение .jsx (а не .txt или .js).
  → Попробуйте запустить через Файл → Сценарии → Другой сценарий...

8.6. Ошибки при работе с цветом "Registration"
  → Если в документе нет цвета "Registration", скрипт автоматически заменит его на "Black".
  → Чтобы добавить Registration: Окно → Образцы → меню → Новый образец → Тип: Регистрационный.


9. ГОРЯЧИЕ КЛАВИШИ
--------------------------------------------------------------------------------
Для быстрого запуска скрипта можно создать операцию (Action):
  1. Окно → Операции (Actions)
  2. Создать новую операцию → назначить функциональную клавишу (например, F2)
  3. Вставить пункт меню: Файл → Сценарии → CropMarks_AI
  4. Остановить запись

Теперь скрипт запускается по нажатию назначенной клавиши.


10. ОБНОВЛЕНИЕ
--------------------------------------------------------------------------------
Для обновления скрипта просто замените файл CropMarks_AI.jsx на новую версию.
Настройки сохраняются отдельно и не будут удалены.
